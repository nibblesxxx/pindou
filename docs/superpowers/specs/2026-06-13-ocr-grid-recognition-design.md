# OCR 网格识别 · 设计规格

## 目标
支持带编号的拼豆图纸自动识别——检测网格 → OCR读取色号编号 → 颜色交叉验证 → 统计扣除。

## 交互流程
1. 上传图片 → 预览
2. 用户点选：背景色 / 主网格色 / 次网格色（容差±10）
3. 自动检测行列数 → 弹窗确认（可手动修正）
4. 预处理：剔除背景+网格线像素
5. 逐格：取主色 + OCR编号 → 交叉验证
6. 展示结果 → 确认扣除

## 技术选型
- Tesseract.js v5 CDN：浏览器端OCR，懒加载
- 网格检测：灰度投影找波谷
- 取色：众数法（排除文字颜色）
- 交叉验证：格子色 vs 编号对应RGB偏差<30

## 容错
- OCR失败格 → 标记？，回退取色匹配
- 整体成功率<70% → 提示可能格式不兼容
- 网格检测失败 → 允许手动输入行列数
